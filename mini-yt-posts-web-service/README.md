# mini-yt-posts-web-service

## Development

- Configure `.env` variables

```bash
cp .env.template .env
```

- Install dependencies

```bash
npm i
```

- Run dev server

```bash
npm run dev
```

## Testing

```bash
npm run test
```

## Build

```bash
npm run build
```

## Production

- Configure env variables (see `.env.template`)

- Run server

```bash
npm start
```
